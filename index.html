Game Leaderboard System (C++)

A C++-based leaderboard management system designed for efficient player score tracking and ranking. The system utilizes Linked Lists for player data storage, a Min-Heap Priority Queue for managing the top 10 scores, and Arrays for quick display. It allows players to submit and update scores, automatically maintaining an accurate and real-time leaderboard with file storage for persistence.

Online Marketplace System (Java)

A Java-based E-Commerce System designed for seamless transactions between buyers and sellers. The system features secure user authentication, a dynamic product catalog, shopping cart functionality, and multiple payment methods (Cash on Delivery, Debit/Credit Card, GCash). Sellers can manage product listings, while customers can browse, add to cart, and checkout efficiently.
